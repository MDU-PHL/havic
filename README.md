# HAVIC
Detect **H**epatitis **A** **V**irus **I**nfection **C**lusters from HAVNET amplicon sequences.  

## Usage

    havic
    havic detect -h
    havic check -h
    havic version -h



## Installation
```
brew install pyqt --with-python3
pip3 install git+https://github.com/schultzm/HAVIC.git
```
